# Global Fashion — Portfolio CV

Single-file 3D portfolio website. No build step, no dependencies.

## Deploy to GitHub Pages

1. Create a repo named `<your-username>.github.io` (or any repo).
2. Upload `index.html` (and this README).
3. Repo → Settings → Pages → Source: `main` branch, `/ (root)` → Save.
4. Site goes live at `https://<your-username>.github.io/` in ~1 minute.

## Customize

Everything is in `index.html` — edit text directly:
- Name/title: the `<h1>` and `.kicker` in the hero
- Experience: the `.t-item` blocks
- Skills: `.skill-card` blocks (`data-w` = bar %)
- Collections: `.flip` cards
- Email: search for `gfashionp@gmail.com`
- Colors: the `:root` CSS variables at the top
